📚 Aide Bovi

Retrouve ici les bases pour comprendre et utiliser Bovi sur ton serveur Discord.

🎫 Tickets

La commande !ticketsetup permet de créer un message avec un bouton. Quand un membre clique dessus, Bovi crée un salon privé entre le membre et le staff.

Dans le ticket, le bouton Fermer le ticket permet de supprimer le salon quand le problème est réglé.

🛡️ Modération

Les commandes de modération permettent de gérer le serveur plus facilement.

Commandes utiles : !clear, !kick, !ban, !unban, !warn, !warnings.

👋 Bienvenue

Bovi peut souhaiter la bienvenue aux nouveaux membres, dire au revoir aux membres qui partent et donner un rôle automatiquement.

Vérifie que les salons bienvenue et au-revoir existent bien sur ton serveur.

⚙️ Permissions

Si une commande ne marche pas, vérifie que Bovi a les bonnes permissions.

Pour les tickets, Bovi doit pouvoir gérer les salons, voir les salons, envoyer des messages et gérer les permissions.

❓ Bovi ne répond pas

Si Bovi ne répond pas, vérifie qu’il est bien en ligne et que tu utilises le bon préfixe.

Bovi utilise les commandes slash /. Exemple : /aide.

🔗 Support

Si tu as besoin d’aide, tu peux rejoindre le serveur support de Bovi avec le bouton en haut de cette page.

Tu peux aussi consulter la page d’accueil pour voir les commandes principales.
