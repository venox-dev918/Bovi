# 🐦 Bovi

**Ton Oiseau, Ton Ami, Ton Bot**

Bovi est un bot Discord simple et pratique, créé pour aider à gérer un serveur Discord plus facilement.

Le bot est encore en développement, mais il possède déjà plusieurs systèmes utiles comme les messages de bienvenue, les messages d’au revoir, les tickets, les rôles automatiques et un panel de configuration.

---

## 🚀 Fonctionnalités

### 👋 Accueil / Au revoir

* Message automatique quand un membre rejoint le serveur
* Message automatique quand un membre quitte le serveur
* Salon personnalisable depuis le panel

### 🎭 Rôle automatique

* Attribution automatique d’un rôle aux nouveaux membres
* Configuration depuis le panel

### 🎫 Tickets

* Système de tickets avec bouton
* Création automatique d’un salon privé
* Bouton pour fermer le ticket
* Choix de plusieurs rôles pouvant voir les tickets

### ⚙️ Panel de configuration

* Connexion avec Discord
* Sélection du serveur à configurer
* Configuration des tickets
* Configuration de l’accueil / au revoir
* Réglages généraux du serveur
